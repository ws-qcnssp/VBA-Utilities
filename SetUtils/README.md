# Utilities for sets
*A collection of simple functions for sets of data*

### Functions
Currently, following functions are implemented in this module:
 - SUMOFMULT
 - SUMOFMULT2
 - SUMOFDIV2
 - SUMOFMARKED
 - AVERAGEOFMARKED
 
### Usage
